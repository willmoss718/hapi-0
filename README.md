# HAPI

First, make sure you have Bun.sh installed. Then clone the repo and run:

```
bun install
bun dev
```

You can then preview changes locally at `localhost:3000`.

Assets to be continually updated are in `src/assets`.

Docs [here](https://docs.google.com/document/d/1gBik_n0XGxph7yjUm4IQdR1A0mNhKZG5ijfjetL9jmg/edit?usp=sharing).
